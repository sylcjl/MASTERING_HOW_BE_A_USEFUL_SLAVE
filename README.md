# MASTERING_HOW_BE_A_USEFUL_SLAVE


### 教學1  如何使用git
 - ##### 上傳修改的檔案
 1. 複製一個git項目
    ```git clone https://github.com/sylcjl/MASTERING_HOW_BE_A_USEFUL_SLAVE.git```
 2. 加入修正一個過的檔案
    ```git add path/to/file```
 3. 為上傳的檔案加入改版的說明
    ```git commit -m "說明內容"```
 4. 上傳
    ```git push -u origin BRANCH_NAME```
